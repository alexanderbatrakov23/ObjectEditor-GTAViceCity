# 🧩 Редактор объектов v1.0 — Custom Menu

Инструмент для управления объектами в сцене: перемещение, вращение, удаление, заморозка персонажа и сохранение состояния.

---

## 🎮 Основное управление

| Клавиша           | Действие                          |
|-------------------|-----------------------------------|
| `M`               | Открыть главное меню              |
| `Y`               | Настройка скорости                |
| `1` / `2`         | Выбрать предыдущий / следующий объект |
| `N`               | Удалить выбранный объект          |
| `H`               | Включить режим редактирования     |
| `C`               | Сохранить объекты в лог           |
| `5`               | Заморозить персонажа*             |
| `6`               | Разморозить персонажа             |

> *\* Используйте, если стандартная заморозка не сработала.*

---

## 🚀 Режим перемещения

| Комбинация        | Действие                          |
|-------------------|-----------------------------------|
| `W` / `S`         | Вперёд / назад                    |
| `A` / `D`         | Влево / вправо                    |
| `Shift` + `W` / `S` | Вверх / вниз                    |

---

## 🔄 Режим вращения

| Комбинация           | Действие                          |
|----------------------|-----------------------------------|
| `Q` / `E`            | Вращение по горизонтали           |
| `A` / `D`            | Вращение по вертикали             |
| `W` / `S`            | Вращение по оси                   |
| `Shift` + `Q` / `E`  | Дополнительный наклон             |

---

## 🕊️ Режим Noclip (свободная камера)

| Клавиша           | Действие                          |
|-------------------|-----------------------------------|
| `Num 8` / `Num 2` | Вперёд / назад                    |
| `Num 4` / `Num 6` | Влево / вправо                    |
| `Num 7` / `Num 9` | Поворот камеры                    |
| `Num 1` / `Num 3` | Наклон вверх / вниз               |
| `Num +`           | Вверх                              |
| `Num -`           | Вниз                               |
| `Num 5`           | Сброс наклона                      |

---

## 📁 Сохранение

- Все объекты сохраняются в **лог** при нажатии `C`.
- Рекомендуется делать это перед удалением объектов или выходом.

---

## ⚠️ Примечание

- Режим редактирования включается клавишей `H` — без него перемещение и вращение могут быть недоступны.
- Заморозка персонажа (`5`) — резервный вариант, если основной механизм не отработал.
